Class EventEmitter<EventMap>

Base class for objects that dispatches events.

example
var emitter = new EventEmitter();
emitter.on('myEvent', function(evt){
    console.log(evt.message);
});
emitter.emit({
    type: 'myEvent',
    message: 'Hello world!'
});
